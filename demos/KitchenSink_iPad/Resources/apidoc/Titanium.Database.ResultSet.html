Titanium.Database.ResultSet

object Titanium.Database
           

The ResultSet instance returned by invoking a database SQL execute.

Methods

Name Description
close.html close the result set and release resources. once closed, this result set must no longer be used
field.html retrieve a row value by field index
fieldByName.html retrieve a row value by field name
fieldCount.html return the number of columns in the result set
isValidRow.html return true if the row is a valid row
next.html iterate to the next row in the result set. returns false if no more results are available

Properties

Name Type Description
rowCount int

the number of rows in the result set
validRow boolean

returns true if the current row is still valid

Events

This object has no events
