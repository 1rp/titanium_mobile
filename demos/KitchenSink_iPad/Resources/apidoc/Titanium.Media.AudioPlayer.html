Titanium.Media.AudioPlayer

object Titanium.Media
           

The AudioPlayer object is returned by Titanum.Media.createAudioPlayer and is used for streaming audio to the device and low-level control of the audio playback.

Methods

Name Description
addEventListener.html add an event listener for the instance to receive view triggered events
fireEvent.html fire a synthesized event to the views listener
pause.html pause playback
removeEventListener.html remove a previously added event listener
setPaused.html control the playback of the audio
setUrl.html change the url of the audio playback
start.html start playback
stateDescription.html convert a state into a textual description suitable for display
stop.html stop playback
