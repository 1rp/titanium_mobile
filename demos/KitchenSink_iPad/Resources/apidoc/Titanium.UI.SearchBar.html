Titanium.UI.SearchBar

object Titanium.UI
           

A Search Bar is created by the method Titanium.UI.createSearchBar.

Methods

Name Description
add.html add a child to the view hierarchy
addEventListener.html add an event listener for the instance to receive view triggered events
animate.html animate the view
blur.html called to force the search bar to lose focus
fireEvent.html fire a synthesized event to the views listener
focus.html called to force the search bar to focus
hide.html hide the view
remove.html remove a previously add view from the view hiearchy
removeEventListener.html remove a previously added event listener
show.html make the view visible
toImage.html return a Blob image of the rendered view

    barColor:'#000', 
    showCancel:true,
    height:43,
    top:0,
});
