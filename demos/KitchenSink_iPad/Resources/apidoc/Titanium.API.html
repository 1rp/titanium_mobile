Titanium.API

submodule of Titanium
           

The top level API module. The API module is mainly used for logging.

Methods

Name Description
addEventListener add an event listener for the instance to receive view triggered events
debug function for logging debug messages
error function for logging error messages
fireEvent fire a synthesized event to the views listener
info function for logging informational messages
log function for logging custom severity messages
removeEventListener remove a previously added event listener
warn function for logging warning messages

Properties

This module has no properties

Events

This module has no events
